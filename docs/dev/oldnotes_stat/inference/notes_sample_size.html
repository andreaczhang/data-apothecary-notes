Two types of statistical inference: confidence intervals, hypothesis testing

Two approaches

  • precision based: what is the sample size required to get the confidence interval for my point estimate (mean, proportion) within a specific range
  • power based: what is the sample size required to detect the minimum clinically relevant difference at a given degree of certainty in a hypothesis test

Source of error

truth + systematic error + random error

systematic error (bias): faulty design, lack of randomization, blinding

random error can be reduced by increasing the sample size

The effect of random error decreases by the square roo of n as samples increases

SE of the mean: sigma/sqrt(n)

SE of a proportion: sqrt(p(1-p)/n)

Sample size calculation

  • expectetd variation in the data (standard deviation)
  • significance level, 5%
  • desired power
  • clinically relevant differences between groups

Precision based

mean

for given precision a, standard deviation sigma, number of observation required:

n = 4 sigma^2 / a^2

proportion

n = 4 p(1-p)/a^2

Hypothesis testing

Power = P(reject h0 | ha true), i.e. identifying a difference between groups if there is a difference

Effect size

formula depends on type of analysis

  • two sample t-test
  • paired -test
  • compare proportions

Effect size

  • clinically relevant difference, \(\Delta\)
  • stadard deviation in both groups, sigma
  • effect size: \(\Delta\)/sigma

Example

  • compare two groups, difference set to be \(\Delta = 0.5\), standard deviation \(\sigma = 1\)
  • Effect size: \(0.5/1 = 0.5\)
  • Set significance level alpha = 0.05, power 1-beta = 0.8
  • (by nomogram) total sample size is 120, i.e. each group needs 60

Calculation

Formula (two means, indep samples)

n = 2 sigma2/delta2 k

  • k = (u+v)^2
  • u: one sided percentage point of normal distribution corresponding to beta
    • u = 0.84 for 80% power
  • v: two-sided percentage point of normal distribution corresponding to alpha
    • v = 1.96 for 5% significance level

(above example)

  • n = 2/0.5^2 *(1.96+0.84)^2 = 62.72

Formula: paired data

cross-over study

effect size: 2 delta/sigma d

  • sigma d is the sd of difference between the two measurements

Proportions in two groups

  • p1, p2
  • relevant difference: p1-p2
  • average proportion: pbar = (p1+p2)/2
  • effect size: (p1-p2)/(sqrt(pbar * (1-pbar)))

Example:

proportions are 0.1, 0.2

average is 0.15

effect size: (0.2-0.1)/sqrt(0.15*(1-0.15)) = 0.28
