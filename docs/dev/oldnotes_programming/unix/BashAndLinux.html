General

/bin directory

it is a standard subdirectory of the root directory / in unix-like OS that contains tthe executables. It contains the most basic programs (ls, rm, …) . Name means binaries, compiled programs. (most important one)

which programName will give where the program is (path). For example, which R will give /usr/local/bin/R and a few other useful programs are under here (psql, brew etc).

/usr stands for Unix System Resources

`` is called ‘backticks’.

View and change permissions

read, write, execute.

ls -l [path]
-rwxr----x
  • first character is file type: - means normal file, d means directory
  • next 3 characters are permissions for owner. if -, absence of permission.
  • next 3 characters are permissions for the group. Read, write and execute. In this example, no permission apart from read.
  • last 3 are permissions for others.
chmod [permission][path]
# example 
chmod g+x frog.png
  • who change? [ugoa]: user (owner), group, others, all
  • granting or revoking the permission? + or -
  • which permission are we setting? r, w, x.

Permission 755 is -rwxr-xr-x, which means standard file -, owner has all permission rwx, group and others have the same permission r-x.

chmod 755 frog.png

select columns

-f for fields to specify columns

-d for delimiter for separator

cut -f 2-5, 8, -d , values.csv    # select col 2 to 5 and 8, using comma as separator.

select with search

select lines according to what file contains

grep keyword file.txt     # returns lines that match keyword 

-c # count of matching lines 
-n # line number
-v # those don't contain
-h # suppress the file names (default when there's only 1 file)
grep -c incisor seasonal/autumn.csv seasonal/winter.csv # count occurence in 2 files

count

wc -l # line count
wc -c # characters,bytes
wc -w # word count

sort

sort -n  # numerically
sort -r # reverse 
sort -b # ignore leading blanks 
sort -f # fold case

Often used with unique,

cut -d , -f 2 seasonal/winter.csv | grep -v Tooth | sort |uniq -c  # give unique and count

redirect output

head -n 5 seasonal/summer.csv > top.csv # saved to top.csv

Note: never in the middle of a pipe!

pipe

head -n 5 seasonal/summer.csv | tail -n 3

Wild cards

*.csv
?017.txt     # single character 
201[78].txt. # either 2017 or 2018. ONLY ONE CHARACTER
{*.txt, *.csv}  # anything inside

loop

for … variable … in .. list .. ; do .. body.. ; done

ITskills andrea$ for filename in *.md; do echo $filename; done  # this gives the names of md files in ITskills

It is convenient to name a bunch of files with one shell variable, then refer to it later on.

files=seasonal/*.csv
for f in $files; do echo $f; done    # the $ in the first part is important
